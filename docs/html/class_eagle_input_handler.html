! Input recording functions for setting Inputs and InputGroups to key/mouse/joystick input

RecordInputPress records the next single Input object, storing it in the provided pointer.

Parameters
inputmust be non-null. RecordInputGroup records the next InputGroup object, storing both the press, and any modifiers held.
input_groupmust be non-NULL These are both blocking calls - they will wait forever until there is input
