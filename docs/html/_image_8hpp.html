enum IMAGE_SOURCE

Stores data about where an image came from.

!

Lets you know whether this image was allocated from memory (any kind), loaded from disk, created from a parent image, whether this is a reference to an image we should not free, or whether we own it

Enumerator
ALLOCATED 

Allocated from main, system, or video memory.

LOADED 

This image was loaded from disk.

SUBBITMAP 

This image is a sub image.

REFERENCE_ONLY 

This image is a reference and should not be destroyed.

OWNIT 

We own this image, no one else should touch it.
