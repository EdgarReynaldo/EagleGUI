Detailed Description

The ColorRegistry class stores colors for use by the gui system and other users.

Pre-defined colors are clear, white, black, light_gray, medium_gray, dark_gray, red, orange, yellow, lime_green , green, neon_green, cyan, sky_blue, blue, purple, magenta, and fuchsia Rather 5 shades of grey, and 12 fully saturated hues

A ColorRegistry object can store both named colors and named colorsets.
