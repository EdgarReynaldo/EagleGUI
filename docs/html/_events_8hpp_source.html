578 /**! @class EagleEventHandler

580  *
581  *
582  * EagleEventHandler's are thread safe, as long as you tell it which thread you're on. For general purposes, and single threaded apps
583  * you can use the default value for an @ref EagleThread*, @ref MAIN_THREAD . The mutexes used need to know which thread they're on
584  * to keep everything synchronized properly.
585  *
586  * An EagleEventHandler enqueues-back events to its message queue (deque) for in order retrieval and event handling.
587  * As an EagleEventListener, it's event handling does several things. One it enqueues the event. Two, at the time of
588  * your choosing, it will emit the event to any objects listening to it. You can set this with the @ref EagleEventHandler::emitter_delay
589  * variable. Pass true to the constructor to delay emitting events until removed from the queue. Pass false to emit events immediately
