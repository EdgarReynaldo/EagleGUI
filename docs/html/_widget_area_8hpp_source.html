19  * @file WidgetArea.hpp
20  * @brief This file contains the interfaces for the different types of widget areas used in Eagle
21  *
22  * Include this header to get access to the @ref BOXAREA, @ref NPAREA, and @ref WIDGETAREA classes
23  *
24  * Eagle widgets use the CSS Box model. See here for details : https://www.w3schools.com/Css/css_boxmodel.asp
25  * Note: A WIDGETAREA includes the margin, border, padding, and inner (client) areas
26  * Note: A BOXAREA only defines the edges of a box, not the box itself, ie. left/right/top/bottom values
27  */
