75 /**! @def EAGLE_ASSERT @def EAGLE_TRACE @def EAGLE_DEBUG
76  *
77  * The EAGLE_ASSERT macro is used just like the C assert function, except that it will log the file, line, function, and expression
78  * used. Does nothing if @ref DEBUG is not defined.
79  *
80  * The EAGLE_TRACE macro is similar, but only does logging. Does nothing if @ref DEBUG is not defined.
81  *
82  * The EAGLE_DEBUG macro will execute code in DEBUG mode only. Be careful with this one, as it can lead
83  * to bizarre differences between release and debug code.
84  *
85  * CREDITS : Thanks to Per Larsson on allegro.cc for this do while trick in the macros
86  */
