55 /*! \class AnimationBase
56  * \brief This class provides functionality common to all animations.
57  *
58  * The AnimationBase class makes animation easy. All you do is tell it the
59  * number of loops, the duration of each loop, and the number of frames per loop
60  * and then you're ready to animate. Call the @ref AnimationBase::Init function to get setup and then
61  * @ref AnimationBase::AdvanceAnimationTime, @ref AnimationBase::SetAnimationTime, or
62  * @ref AnimationBase::SetAnimationPercent to alter the playback time.
63  *
64  * When you need to find out the normalized animation percent or frame number, call
65  * @ref AnimationBase::GetFrameNum or @ref AnimationBase::GetNormalizedPercent. The frame number
66  * is equivalent to :
67  *
68  * (int)(GetNormalizedPercent()*GetNumFrames())
69  *
70  */
