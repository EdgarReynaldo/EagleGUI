Include this header to get access to the BOXAREA, NPAREA, and WIDGETAREA classes

Eagle widgets use the CSS Box model. See here for details : https://www.w3schools.com/Css/css_boxmodel.asp Note: A WIDGETAREA includes the margin, border, padding, and inner (client) areas Note: A BOXAREA only defines the edges of a box, not the box itself, ie. left/right/top/bottom values
