Detailed Description

Tree class for logical grouping of Input evaluations and storage.

!

How to implement the && || ! logic and chaining? With an object tree

Trying to get a bool value from an empty input group will throw an exception, as returning true or false would be incorrect, since there's no logical test to evaluate
