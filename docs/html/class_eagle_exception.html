Detailed Description

Base exception class for Eagle.

!

This class is the basic class for all Eagle exceptions. It automatically logs errors to EagleError and will intentionally crash on a failed assert given DEBUG and EAGLE_ASSERT_EXCEPTION are defined so you can get a backtrace.

#define DEBUG
#define EAGLE_ASSERT_EXCEPTION
#include "Eagle/Exception.hpp"
EAGLE_ASSERT(0);///< Halt and log in debug mode
throw EagleException("Halt in debug or release mode with an exception");
