◆ WIDGETCOLOR

enum WIDGETCOLOR

Enumerates the standard color scheme for a widget and its corresponding place in a WidgetColorset.

WIDGETCOLOR ranges from darkest (SDCOL shadow color) to lightest (HLCOL highlight color), with a few extras for hover, and the CSS box model's margin, border, and padding areas.
