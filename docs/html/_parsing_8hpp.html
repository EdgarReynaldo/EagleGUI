EagleColor ParseColor ( const std::string  color_dec)

Parses a color from text.

!

Valid attributes for an EagleColor are RGB, RGBA, FRGB, and FRGBA. Value takes form of parenthesis enclosed comma separated list of values. Ex. EagleColor lavender = RGB(127,64,255) Declaration follows format [RGB | RGBA | FRGB | FRGBA](n,n,n[,n]) eg. RGB(255,255,255) or FRGBA(1.0,1.0,1.0,1.0) Valid values for RGB and RGBA are integers from [0,255] Valid values for FRGB and FRGBA are floats or doubles from [0.0,1.0]

Declaration follows format [RGB | RGBA | FRGB | FRGBA](n,n,n[,n]) eg. RGB(255,255,255) or FRGBA(1.0,1.0,1.0,1.0)
