Detailed Description

A simple class to create colors.

!

EagleColor has a string constructor. The format is one of the following : "RGB %i,%i,%i" OR "RGBA %i,%i,%i,%i" OR "FRGB %f,%f,%f" OR "FRGBA %f,%f,%f,%f"

i represents a textual integer, and f represents a textual float
