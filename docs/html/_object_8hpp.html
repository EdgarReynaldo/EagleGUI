EAGLE_ID

Typedef for the eagle id type. Retrieve any EagleObject from the ObjectRegistry by using its EAGLE_ID.

!

All EAGLE_IDs are unique, starting from zero

The only invalid EAGLE_ID is EAGLE_ID_INVALID, but all ids greater than or equal to CurrentId are out of range
